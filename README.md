# Atelier - Subscription Tracker

A rich, aesthetic subscription tracking UI built entirely with Vanilla HTML, CSS, and JS (No libraries, no Tailwind).
Based on the "Financial Atelier" design system.

## Features
- Dashboard with total spending and upcoming renewals.
- "Active Portfolio" list of subscriptions.
- Analytics page showing breakdown of spend.
- Subscription details view.
- Complete Vanilla CSS styling with glassmorphism, depth, and custom properties.
- Dynamic page switching via Vanilla JS.

## Run Locally
Just open `index.html` in any modern web browser or use a local HTTP server:
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000`
